BAM2MID is a command-line utility to convert BAM (Bob's Adlib Music) format
files into MIDI format files. (the looping features of BAM will be lost)

USAGE:

   bam2mid filename.bam

Or you can drag your bam file and drop it on bam2mid.exe

The source code is dual licensed under the GNU GPL v2+ and MIT Licenses. See LICENCE.txt
It can be downloaded from http://rpg.hamsterrepublic.com/ohrrpgce/Source
To compile, install FreeBasic and SConscript, and run scons bam2mid
